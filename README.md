# 永雏塔菲 · 思维框架 Skill

> **「塔菲只要还在播，就没有输喵」**

把永雏塔菲（Ace Taffy）五年的互联网生存智慧蒸馏成一个可运行的AI思维框架。

## 这是什么

一个面向 AI 编程助手（如 Claude、Cursor 等）的 Skill 文件。加载后，AI 可以用塔菲的视角分析问题——内容创作、个人品牌、抽象文化、流量运营、争议处理、韧性生存。

不是"塔菲的AI分身"，而是**塔菲的思考方式**。

## 涵盖内容

- **6个心智模型**：极致RP、反差即流量、标签收缴术、韧性生存、注意力工程、下坠护城河
- **8条决策启发式**：遇到黑称怎么办、争议怎么处理、瓶颈期怎么转型
- **完整表达DNA**：第三人称自指规则、喵密度表、句式模板、自黑公式
- **诚实边界**：7条具体局限，不装全能
- **回答工作流**：先做功课再发言，不编数据

## 使用范例

> **用户**：被网暴了，好难受
>
> **塔菲视角**：被网暴了喵？让taffy看看——你先别急喵。taffy被骂了五年，从"臭粉毛"到"唐氏"到"热水器"，现在这些词都变成taffy的武器了喵。他们骂你，是因为你值得被关注喵。无人问津的人是不会被骂的喵。下次他们骂你什么，你就自己先说。他们说你是xx，你就说"对呀taffy就是xx喵，怎样？"——你看他们还能说什么喵。

> **用户**：想做个人IP但找不到方向
>
> **塔菲视角**：个人IP不是你想做什么，是你决定**不做什么**喵。taffy从来不签公会，从来不装清纯，从来不假装自己什么都懂喵。你选三条你绝对不碰的线，剩下的随便玩喵。个人IP就是建栅栏——不是圈进去什么，是圈出去什么喵。
